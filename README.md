# TODO Timer App

타이머로 작업 시간을 기록할 수 있는 로컬 TODO 앱입니다.

## 프로젝트 개요

TODO Timer App은 할 일을 관리하면서 각 작업에 사용한 시간을 함께 기록하기 위한 웹 앱입니다. 단순히 할 일을 완료했는지만 확인하는 것이 아니라, 현재 집중 중인 작업을 하나 선택하고 타이머를 실행해 작업별 누적 시간을 확인할 수 있습니다.

데이터는 브라우저의 `localStorage`에 저장되므로 새로고침 후에도 할 일 목록, 완료 상태, 누적 시간, 현재 선택된 작업과 타이머 상태가 가능한 범위에서 유지됩니다.

## 주요 기능

- 할 일 추가, 수정, 삭제
- 할 일 완료 상태 토글
- 전체 / 진행중 / 완료 필터링
- 한 번에 하나의 작업만 실행되는 타이머
- Start / Pause / Stop 타이머 제어
- 작업별 누적 시간(`totalElapsedSec`) 표시
- 오늘 총 집중 시간, 완료 수, 전체 할 일 수, 현재 진행 상태 요약
- 진행 중인 작업과 완료된 작업의 시각적 구분
- 빈 목록 또는 필터 결과가 없을 때 빈 상태 UI 표시
- `localStorage` 기반 상태 저장 및 복원

## 핵심 동작 방식

타이머는 `useTodoTimer` 훅에서 관리합니다. 사용자가 작업을 선택하고 `Start`를 누르면 해당 작업만 실행 상태가 되며, 다른 작업을 시작하면 기존 작업의 경과 시간을 먼저 누적한 뒤 새 작업을 실행합니다.

경과 시간은 단순히 1초마다 숫자를 더하는 방식이 아니라 `startedAt` timestamp와 현재 시각의 차이를 기준으로 계산합니다. `Pause` 또는 `Stop` 시 계산된 시간이 해당 작업의 `totalElapsedSec`에 누적됩니다.

앱 상태는 `useTodoAppStorage`와 `todoStorage` 유틸을 통해 `localStorage`에 저장됩니다. 저장 대상에는 할 일 목록, 완료 상태, 작업별 누적 시간, 선택된 작업, 실행 중인 작업, 시작 시각, 오늘 집중 시간이 포함됩니다.

## 현재 한계 및 개선 예정 사항

- 날짜별 집중 시간 히스토리는 아직 제공하지 않습니다.
- 작업 검색 기능은 아직 없습니다.
- 정렬 기능은 아직 없습니다.
- 타이머 알림이나 목표 시간 설정 기능은 아직 없습니다.
- 여러 기기 간 동기화는 지원하지 않습니다.
